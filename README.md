# Bitpad

A developer-focused text editor for notes, ideas, and inspiration built with PySide6.

## Installation

1. Clone or download this repository
2. Install PySide6:
```bash  
pip install PySide6
```
3. Run the application:
```bash
python main.py
```
## Keyboard Shortcuts

| Action | Shortcut |
|--------|----------|
| New Tab | Ctrl+N |
| Open File | Ctrl+O |
| Save | Ctrl+S |
| Save As | Ctrl+Shift+S |
| Undo | Ctrl+Z |
| Redo | Ctrl+Y |
| Find | Ctrl+F |
| Find & Replace | Ctrl+H |
| Exit | Ctrl+Q |
